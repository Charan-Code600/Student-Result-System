


# 🎓 Student Result System

A Python based student result management system with CSV storage.

## Features
- ➕ Add Student with 5 subject marks
- 📋 Show All Results
- 🔍 Search by Name or Roll Number
- 🏆 Class Topper
- 📊 Class Summary (Pass/Fail count)
- 🗑️ Delete Student
- 💾 Auto save to CSV

## How to Run
```bash
python student_result_system.py
```

## Technologies Used
- Python
- Pandas
- CSV
